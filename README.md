#Weather App#

A simple demo of weather forecast in Australia.

##Screenshot##
![Weather App](/screenshot.png?raw=true)


##Technologies involved##

* Angular
* Bootstrap
* Grunt
* Karma
* Jasmine
* JSHint
* Express
* Node.js

##How to run##

1. npm install
2. cd public && bower install
3. cd ..
4. node server.js
5. go to 127.0.0.1:3000
